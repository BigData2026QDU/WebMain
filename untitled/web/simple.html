极简使用示例

这个页面只引入了一份 JS 文件，无需编写任何 JavaScript 代码即可完成日间/夜间主题切换。

主题会根据时间自动切换，右上角的按钮也会自动生成。

使用方法

在 HTML 中引入主题管理器即可：

<script src="js/theme-manager.js"></script>

无需编写额外代码。

自定义配置（可选）

通过 data 属性传递配置：

<script 
  src="js/theme-manager.js"
  data-day-start="7"
  data-night-start="19"
  data-interval="30000"
></script>

可用配置项：

  • data-day-start - 日间模式开始时间（小时，默认 6）
  • data-night-start - 夜间模式开始时间（小时，默认 18）
  • data-interval - 自动检测间隔（毫秒，默认 60000）
  • data-storage-key - localStorage 键名（默认 'theme-preference'）
  • data-manual-override - 是否允许手动覆盖（默认 true）
  • data-auto-init - 是否自动初始化（默认 true）
  • data-auto-button - 是否自动创建按钮（默认 true）
  • data-button-selector - 自定义按钮选择器（可选）

高级用法

如需通过 JS 控制主题，可访问全局 window.themeManager 对象：

// 获取当前主题
console.log(themeManager.getCurrentTheme());

// 手动切换主题
themeManager.toggleTheme();

// 设置特定主题
themeManager.setTheme('night');

// 清除保存的偏好，恢复自动模式
themeManager.clearSavedTheme();

使用自定义按钮

如果你想使用自定义按钮，而不是自动生成的按钮：

<button id="myThemeButton">切换主题</button>

<script 
  src="js/theme-manager.js"
  data-button-selector="#myThemeButton"
></script>

前后端通信演示

点击按钮，模拟一次 API 请求。请求状态将显示在右下角的胶囊中。

报告渲染解码器

从服务器获取编码的报告内容，解码后动态渲染文字、指标、图表，全程使用预定义的主题颜色。
