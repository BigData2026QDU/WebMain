HBase表设计文档
==================

表名: student_comprehensive

RowKey设计: Grade_Gender_StudentID
- 格式示例: 10_Male_00001, 11_Female_00523
- 设计理由:
  1. 按年级分区，便于按年级范围查询
  2. 性别作为第二维度，支持性别分析
  3. StudentID保证唯一性

列族设计:
---------
1. 列族: basic (基本信息)
   - basic:age
   - basic:race
   - basic:ses_quartile
   - basic:parental_education

2. 列族: academic (学业信息)
   - academic:test_score_math
   - academic:test_score_reading
   - academic:test_score_science
   - academic:gpa

3. 列族: behavior (行为信息)
   - behavior:attendance_rate
   - behavior:study_hours
   - behavior:extracurricular
   - behavior:part_time_job

4. 列族: environment (环境信息)
   - environment:school_type
   - environment:locale
   - environment:internet_access
   - environment:parent_support

5. 列族: lifestyle (生活方式)
   - lifestyle:romantic
   - lifestyle:free_time
   - lifestyle:go_out

HBase Shell创建命令:
--------------------
create 'student_comprehensive',
  {NAME => 'basic', VERSIONS => 1},
  {NAME => 'academic', VERSIONS => 1},
  {NAME => 'behavior', VERSIONS => 1},
  {NAME => 'environment', VERSIONS => 1},
  {NAME => 'lifestyle', VERSIONS => 1}
