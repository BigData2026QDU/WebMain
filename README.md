# hivehbase

#### 介绍
这是一个大数据学期项目。

#### 技术选型
1. Apache ECharts 图表
2. Java Servlet 网页控制器
3. Html 静态网页
4. Axios 异步通信
5. Jackson JSON传输
6. Gsap 动画制作
7.  **OpenRefine 数据清洗**
